## Github flow
1. Clone Repositorie
2. Run `npm install` (use node version 14.18.2)
3. Run `npm start`

Implement landing page according to [Figma design](https://www.figma.com/proto/dBmGAtYTwFK8H95fGHX2Qo/Promodex-Partner-Program?node-id=2%3A3) - Use BEM and SCSS
- The design 1728px
- Notebook 1280px
- Tablet 768px
- Mobile (> 320px)
